# 📅 Jump Table — AUTO SCHEDULER v1.0

대학교 수업 시간표를 자동으로 생성·추천해주는 단일 HTML 웹 앱입니다.  
별도 설치나 서버 없이 브라우저에서 바로 실행됩니다.

---

## 주요 기능

- **수업 등록** — 수업명, 교수명, 요일, 시작교시, 강의 시간(교시 수), 학점, 수업 유형(전공/교양/기타), 필수 여부를 입력
- **자동 시간표 생성** — 등록된 수업 조합에서 조건에 맞는 시간표를 최대 7개 추천
- **조건 설정** — 공강 요일 희망, 최소·최대 학점, 연강 제한, 점심시간 보장 등 설정 가능
- **가중치 조정** — 공강·점심·연강·학점 항목별 우선순위 슬라이더로 세밀한 제어
- **결과 탭** — 각 후보 시간표의 적합도 점수, 공강 요일, 점심 보장 여부, 연강 수 등 요약 표시
- **시각적 시간표** — 요일별 색상으로 구분된 시간표 테이블 렌더링
- **테마 설정** — 라이트/다크 모드, 액센트 컬러 변경, 배경 효과·모션 감소 옵션
- **개구리 마스코트** — 시간표 생성 시 등장하는 애니메이션 마스코트 🐸
- **토스트 알림** — 수업 추가·삭제·초기화 시 하단 알림 표시 (설정에서 끄기 가능)

## 사용 방법

1. `JumpTable.html`을 브라우저로 열기
2. **수업 입력** 탭에서 수업 정보 입력 후 `수업 추가` 버튼 클릭
3. **시간표 생성** 탭에서 조건(공강, 학점, 연강 등) 설정
4. `✨ 시간표 생성` 버튼 클릭
5. **결과** 탭에서 추천 시간표 확인 및 비교

---

## 알고리즘 개요

| 단계 | 설명 |
|------|------|
| 조합 생성 | 등록된 수업에서 시간 충돌 없는 조합을 탐색 |
| 필수 수업 | `필수` 체크된 수업은 모든 후보에 반드시 포함 |
| 점수 산정 | 공강·점심·연강·학점 가중치 기반 종합 점수 계산 |
| 정렬 및 출력 | 상위 7개 후보를 점수 순으로 탭 형태로 출력 |

## CSS 레이아웃 구조

수업 과제 조건을 충족하는 레이아웃 기법이 적용되어 있습니다.

- **float + overflow: hidden** — 헤더 수평 정렬 및 One True 레이아웃
- **CSS Grid** — 입력 폼 2·3열 배치
- **CSS 변수 (`--lake`, `--sage` 등)** — 테마 컬러 시스템
- **다크 모드** — `body.dark` 클래스 토글로 전환
